核心信息 · MESSAGE

清楚 = 一句话能被回家路上的人复述

“Chat is a feature. UI is the product.
Don't bolt on. Redesign.” 这一句必背逐字 · S13 与 S22 完全一致

敌人不是"含糊",是载体(L2–L6 阶梯)抢了信息的风头。让阶梯永远服务于那句话:

  1. 做减法。 删掉那 4 张 Salesforce/趋势 slide — 它们稀释主线、违背"零厂商推销"。
  2. 每讲完一层都给载体降级。 固定收尾句:“That's not the point — the point is you CHOSE this, on purpose, instead of bolting on chat.”
  3. "because" 是承重墙。 让听众说 “Feature X should be L-_ because [a question]” — L 编号是载体,because 才是信息。
  4. 回环要逐字。 S22 分隔线下方那行字 = S13 那行,别换字(现在换了,识别火花会灭)。
